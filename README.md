2. my_key.pem (Private key of Certificate - BEGIN RSA PRIVATE KEY - ...)
3. my_ca.cert (-----BEGIN CERTIFICATE----- ... -----END CERTIFICATE----------BEGIN CERTIFICATE-----...-----END CERTIFICATE-----)
